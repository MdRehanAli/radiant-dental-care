# Quick Start Guide

## 1. Install Dependencies

```bash
npm install
```

## 2. Install Tailwind Plugins

```bash
npm install -D @tailwindcss/forms @tailwindcss/container-queries
```

## 3. Start Development Server

```bash
npm run dev
```

## 4. Open in Browser

Visit [http://localhost:3000](http://localhost:3000) to see your app running.

## Project Overview

The app consists of:

- **Header** - Fixed navigation bar with logo and CTA button
- **Hero** - Large hero section with headline, image, and rating badge
- **WhyChooseUs** - 3-column card grid showcasing key features
- **Testimonials** - Patient testimonials with ratings and avatars
- **CTA** - Call-to-action section with gradient background
- **Footer** - Footer with links, contact info, and social icons

All components are built with:
- React (JSX)
- Tailwind CSS for styling
- Material Design 3 colors
- Material Symbols for icons
- Responsive design
